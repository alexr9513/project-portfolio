About This Project

This project combines shell scripting, data processing, and web visualization to create an automated cryptocurrency monitoring system. The solution scrapes real-time cryptocurrency data from leading platforms, stores it efficiently, and presents it through an interactive dashboard for analysis and decision-making.

Key Components

  • Data Scraping: Bash script with regular expressions to extract cryptocurrency data from CoinGecko and CoinCodex
  • Data Storage: Structured CSV format for efficient storage and retrieval of time-series cryptocurrency data
  • Visualization Dashboard: Interactive Plotly-powered interface displaying price trends, volume analysis, and market correlations
  • Automation: Scheduled hourly updates (8 AM to 6 PM) for continuous monitoring during active trading hours

Technical Implementation

  • Bash Scripting: Custom regular expressions for targeted data extraction from cryptocurrency websites
  • Data Processing: Cleaning and structuring scraped data for analysis and visualization
  • Dashboard Development: Plotly visualization library for creating interactive charts and metrics
  • Cloud Deployment: Running on AWS EC2 instance for reliable 24/7 operation
  • Task Scheduling: Cron jobs for automated hourly data updates

Results & Impact

The dashboard provides real-time insights into cryptocurrency market trends, enabling data-driven investment decisions. The automated nature of the system ensures consistent data collection without manual intervention, while the cloud deployment ensures high availability and accessibility from anywhere.
